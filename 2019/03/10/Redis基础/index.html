  • 后台启动redis

    1. 修改 /etc/redis.conf 配置文件, bind 0.0.0.0
    2. 修改配置文件的 requirepass password , 添加密码
    3. 重启redis服务

五种数据库类型

  • 字符串(string)

    可以是字符串、整数或浮点数

    1. set: 设置存储值
    2. get: 获取存储值
    3. del: 删除指定的键值

  • 列表(list)

    一个字符串的链表

    1. rpush: 右侧插入
    2. lpush: 左侧插入
    3. lrange: 列出指定位置区间的值
    4. lindex: 查询指定位置上的值
    5. rpop: 右侧弹出
    6. lpop: 左侧弹出

  • 集合(set)

    一个字符串的无序不重复的集合

    1. sadd: 添加到集合
    2. smember: 查询所有元素
    3. sismember: 检查指定元素是否在
    4. srem: 移除指定元素

  • 散列(hash)

    包含键值对的无序散列

    1. hset: 设置键值
    2. hget: 获取值
    3. hgetall: 获取全部键值
    4. hdel: 删除键值

  • 有序集合(zset)

    有序的set

    1. zadd: 添加元素
    2. zrange: 列出区间内元素
    3. zrangebyscore: 列出score区间内元素
    4. zrem: 移除元素
